# Source Ledger

This companion file tracks which ancient texts, themes, and original additions inform each Illumination of *The Book of Flow*.

This project is not a translation. It is a canonical, non-dualistic mythic synthesis inspired by the Nag Hammadi Library and related wisdom traditions.

## How to Use This Ledger

For each Illumination, track:

- **Primary Sources** — ancient texts that directly inform the section.
- **Secondary/Related Sources** — texts or traditions that echo the themes.
- **Core Themes** — major theological, mythic, psychological, or symbolic ideas.
- **Original Additions** — narrative bridges, dialogue, and mythology created for *The Book of Flow*.
- **Contemplation Question** — the question placed at the end to keep the reader pondering.
